# NewTube

A work-in-progress video platform built with Next.js.

## Status

This project is **not finished yet** and is still under active development. Expect breaking changes.

## Getting started

### Prerequisites

- Bun (recommeneded) or Node.js

### Install dependencies

```bash
npm install
# or
bun install
```

### Run the dev server

```bash
npm run dev
# or
bun run dev
```

## Scripts

- `dev` - start Next.js in development mode
- `dev:all` - run the app + webhook tunnel (requires `ngrok`)
- `build` - production build
- `start` - run the production server
- `lint` - run linting

## Notes

- Environment variables live in `.env.local` (see the repo for an example).
